panchanga API

Drik Panchanga (Smārta, pūrṇimānta) — tithi, nakṣatra, festivals & eclipses, as JSON.

Endpoints

All are GET, return JSON, and accept a location as either a place preset or explicit lat, lng & tz.

  • /api/panchanga?date=YYYY-MM-DD&place=calgary — the day's pañcāṅga
  • /api/festivals?year=2026&place=calgary — the year's festivals
  • /api/eclipses?year=2026&place=calgary — grahaṇas (eclipses)
  • /api — machine-readable usage & the list of place presets

Examples

curl "https://YOUR-DEPLOYMENT/api/festivals?year=2026&place=calgary"
curl "https://YOUR-DEPLOYMENT/api/panchanga?date=2026-11-08&lat=51.04&lng=-114.07&tz=America/Edmonton"
